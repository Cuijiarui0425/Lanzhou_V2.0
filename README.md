# 高精度球盘摩擦多轴异构测控系统

## 1. 项目简介 
本项目为特种高精度球盘摩擦试验机开发的**异构多轴中央主控系统 (Heterogeneous Multi-axis Master Control System)**。在整体软硬协同架构中，主控节点向下接管研华 PCIE-1805 高频数据采集卡（1MS/s 模拟量采集），向外通过 Modbus TCP 协议栈与基于 EtherCAT 总线的底层 PLC 节点及伺服电机集群进行确定性闭环交互。系统摒弃了传统的串行阻塞开发模式，对标 RTOS 任务调度理念，构建了多线程高吞吐并发架构，实现了多轴高精度运动控制、动态力/位混合闭环以及海量传感器数据的实时解算与零丢包落盘。

## 2. 项目指标要求 (Technical Requirements & Metrics)
根据试验机的严苛机械物理约束与测控标准，系统需达到以下核心技术指标：
* **极限转速与同步控制**：球电机额定转速 $\ge$ 6000 rpm，盘电机额定转速 $\ge$ 3000 rpm，需实现极低延迟的独立调速与强同步控制。
* **零累计误差定位**：在极小角度（如正反转 30°）的高频往复摆动工况下，经长时间多周期循环后，机械位置必须实现**零累计误差 (Zero Cumulative Error)**。
* **高频实时闭环与计算**：
  * **采样率**：底层拉压传感器模拟量需达到 1MS/s 的极限高频采集。
  * **物理解算**：主控周期内需实时动态完成卷吸速度、滑滚比 (SRR) 与牵引力系数的高速浮点解算。
* **力/位混合闭环与动态轨迹**：实现基于离散时间间隔与目标载荷的**多阶动态轨迹规划（阶梯变速变载）**；下压丝杠需根据压力传感器反馈实现毫秒级的力矩闭环与自适应加载/卸载。
* **系统级安全限制**：需具备处理硬件越限中断的能力，实现底层物理限位触发瞬间的急停锁死及点动脱困软硬互锁机制。

## 3. 主要工作 (Main Contributions)
作为主控端软硬协同核心开发者，我主导了以下底层架构设计与核心功能研发：

* **构建 1MS/s 高频数据流引擎与并发解耦机制**
  深度调用 PCIe 底层驱动封套，针对 1MS/s、16-bit 双通道极高频采集中断，设计了基于**内存锁与无锁环形缓冲区 (Lock-free Ring Buffer)** 的生产者-消费者模型。利用高吞吐跨线程通信通道 (`System.Threading.Channels`)，将高优先级 DAQ 采集中断、Modbus 确定性通信与异步 I/O 落盘彻底解耦，确保在 1000Hz 操作系统任务调度下实现**实时数据流零丢帧 (Zero Packet Loss)** 与系统级防阻塞。
* **全栈实现确定性通信总线与柔性状态机 (FSM)**
  独立解析 Modbus TCP 底层协议栈，处理大/小端字节序翻转与跨架构内存对齐。构建跨设备共享内存映射寄存器表 (Memory-Mapped Registers)，设计了涵盖 `Idle -> Loading -> Running -> EStop` 的多阶队列式状态机。引入纳秒级心跳包防断线与指令读回握手机制，构建防电机“飞车”的软硬急停互锁防线。
* **实现动态力/位混合闭环与防误差算法**
  针对丝杠电机的下压与卸载动作，动态切换位置环与转矩环，处理传感器延迟与电机惯量的耦合。针对高频往复运动，接管底层编码器位置反馈，通过绝对位置标定与指令补偿算法，彻底消除机械长时间运行下的累计误差。
* **部署实时数字信号处理 (DSP) 与降噪算法**
  面对高频电机震动导致的底层模拟量零点漂移问题，在主控调度层开发定制化 **N 点滑动平均 (Moving Average)** 与毛刺滤除抗混叠算法。将微弱模拟信号的噪声有效抑制，保证了微牵引力与摩擦系数动态解算的高精度与稳定性。
